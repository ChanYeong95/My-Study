## Q. 잘 만들어진 VO란 무엇인지 기술하시오.

1. 정보은닉(Information Hiding)
2. 기본 생성자(Default Constructor)를 명시적으로 작성
3. Getter, Setter를 통해 Private 멤변수에 접근
4. toString 재정의(Override)를 통해 객체의 값을 출력
